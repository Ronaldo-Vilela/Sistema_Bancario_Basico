# Sistema Bancário Básico

Um sistema bancário simples desenvolvido em Python utilizando Programação Orientada a Objetos (POO).

## 📋 Descrição

Este projeto foi desenvolvido como desafio prático durante o Bootcamp **Luizalabs - Back-end com Python - 2º Edição** da [Digital Innovation One (DIO)](https://www.dio.me/).

O sistema implementa operações bancárias básicas como:
- Criação de contas
- Depósitos
- Saques
- Consultas de saldo
- Histórico de transações

## 🛠️ Tecnologias

- **Python** - Linguagem de programação
- **POO** - Programação Orientada a Objetos

## 📁 Estrutura do Projeto

```
Sistema_Bancario_Basico/
├── sistena-bancario.py    # Código principal do sistema
└── .gitignore             # Arquivos ignorados pelo git
```

## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/Ronaldo-Vilela/Sistema_Bancario_Basico.git
```

2. Navegue até o diretório do projeto:
```bash
cd Sistema_Bancario_Basico
```

3. Execute o programa:
```bash
python sistena-bancario.py
```
